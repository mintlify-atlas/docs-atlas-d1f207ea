<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get your portfolio up and running in under 5 minutes</p>
  
  <Steps>
    <Step title="Clone the repository">
      Clone the portfolio repository to your local machine.
      
      ```bash
      git clone https://github.com/xItzHypeR/portfolio.git
      cd portfolio
      ```
    </Step>
    
    <Step title="Install dependencies">
      Install all required packages using npm, yarn, or pnpm.
      
      <CodeGroup>
      ```bash npm
      npm install
      ```
      ```bash yarn
      yarn install
      ```
      ```bash pnpm
      pnpm install
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Start development server">
      Launch the Vite development server with hot module replacement.
      
      ```bash
      npm run dev
      ```
      
      <Note>The app will be available at `http://localhost:5173` and on your local network.</Note>
    </Step>
    
    <Step title="Customize your content">
      Update the projects data, personal information, and styling to make it your own. See the [Customization](/customization/content) guide for details.
    </Step>
  </Steps>
</div>
